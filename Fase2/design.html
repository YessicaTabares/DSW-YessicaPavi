Sitio web para el aprendizaje de etiquetas básicas de HTML,atributos y valores

El proposito de la siguiente página tiene como finalidad adquir conocimientos y estudiar la etiquetas básicas, atributos
y valores que pérmiten la creación de páginas web. La importancia de conocer cada elemento de HTML radica en que
sin estas no podriamos navegar ¿porqué no podriamos navegar? sin etiquetas, atributos y valores seria muy complicado
comprender lo que se quiere transmitir, es decir, no se comprenderia el elemento creado.

Etiquetas básicas de HTML

Existen una serie de etiquetas que son las más usadas para crear cualquier documento HTML,
a continuación las explicamos:

Etiqueta
Función
HTML
 Indica el comienzo del documento HTML
HEAD
 Indica la cabecera de la página
BODY
 Es el cuerpo de la página, en donde
va lo que se se ve en el navegador al cargar
H1,H2...
 Son los titulos o encabezados,siendo h1 el mas grande
y el h6 el más pequeño
A
 Define enlaces
P
 El texto de esta etiqueta forman un parrafo
B
 Se utiliza para resaltar el texto
U
 Texto subrayado
I
 Texto en cursiva
MARQUEE
 Da movimiento al texto

Atributos y Valores

Las etiquetas son la estructura básica del HTML. Estas etiquetas o tags se componen y contienen
otras propiedades, como son los atributos y el contenido.
Una etiqueta por sí sola a veces no contiene la suficiente información para estar completamente
definida. Para ello contamos con los atributos: pares nombre-valor separados por "=" y escritos en
la etiqueta inicial de un elemento después del nombre del elemento. El valor puede estar encerrado
entre "comillas dobles" o 'simples'.
