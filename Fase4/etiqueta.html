¿Qué es HTML?

HTML (por sus siglas en ingles, HyperText Markup Language), es un lenguaje de marcado que nos permite hacer la estructura de una página web. Tenemos que aclarar que HTML no es un lenguaje de programación, pues carece de mecanismos para ejecutar cálculos, repeticiones o condiciones, lo que sí es, es un lenguaje de marcado. Es un lenguaje que le indicará al navegador web qué tipo de elemento es el que está en la estructura de una página web.


Que necesitamos para programar con HTML



1. Para empezar a escribir código HTML necesitamos un editor de código como puede ser: sublime text, notepad, visual studio code, CoffeCup Free Editor, entre otros e incluso el famoso bloc de notas. Para escribir código HTML nos servirá hasta el editor más básico y sencillo.

2. Necesitamos un navegador que nos sirva para visulizar el resultado de nuestro código HTML en la pantalla

3. creamos un documento HTML debemos guardarlo con la extensión .html para que el navegador pueda ejecutarlo como tal.

Etiquetas básicas de HTML



Existen una serie de etiquetas que son las más usadas para crear cualquier documento HTML,

Tabla 1. Etiquetas básicas HTML

Etiqueta
Función
HTML
Indica el comienzo del documento HTML
HEAD
Indica la cabecera de la página
BODY
Es el cuerpo de la página, en donde va lo que se se ve en el navegador al cargar
H1,H2...
Son los titulos o encabezados,siendo h1 el mas grande y el h6 el más pequeño
A
Define enlaces
P
El texto de esta etiqueta forman un parrafo
B
Se utiliza para resaltar el texto
U
Texto subrayado
I
Texto en cursiva
MARQUEE
Da movimiento al texto
