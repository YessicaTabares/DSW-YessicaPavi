Propósito del sitio web


Tiene como finalidad adquir conocimientos y estudiar la etiquetas básicas, atributos y valores que pérmiten la creación de páginas web. La importancia de conocer cada elemento de HTML radica en que sin estas no podriamos navegar ¿porqué no podriamos navegar? sin etiquetas, atributos y valores seria muy complicado comprender lo que se quiere transmitir, es decir, no se comprenderia el elemento creado.



Etiquetas HTML



HTML es un markup language, lo que significa que está escrito con códigos que puede leer una persona sin que sea necesario compilarlo primero. En otras palabras, el texto en una página web está «marcado» con estos códigos para dar instrucciones al navegador web sobre cómo mostrar el texto. Estas etiquetas de marcado son las propias etiquetas HTML.



Atributos y Valores



Las etiquetas son la estructura básica del HTML. Estas etiquetas o tags se componen y contienen otras propiedades, como son los atributos y el contenido. Una etiqueta por sí sola a veces no contiene la suficiente información para estar completamente definida. Para ello contamos con los atributos: pares nombre-valor separados por "=" y escritos en la etiqueta inicial de un elemento después del nombre del elemento. El valor puede estar encerrado entre "comillas dobles" o 'simples'.
